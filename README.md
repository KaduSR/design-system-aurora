# Design System Aurora 🎨

> Design System completo com 40+ componentes, tokens multi-tema e acessibilidade WCAG AA.

## Sobre

Design System profissional com componentes React documentados no Storybook, 
design tokens gerenciados via Style Dictionary, suporte a tema dark/light e 
testes de acessibilidade automatizados com axe-core.

## Stack

React 18 • Storybook 8 • TypeScript • Style Dictionary • axe-core

## Estrutura

packages/
  tokens/     → Design tokens (cores, tipografia, spacing)
  react/      → Componentes React
docs/         → Storybook

## Funcionalidades
- 40+ componentes reutilizaveis
- Tema dark/light por tokens
- Documentacao interativa (Storybook)
- Testes de acessibilidade automatizados
- Design tokens via Style Dictionary
- Bundle otimizado (tree-shakeable)

## Metrica
100% dos componentes aprovados em auditoria WCAG AA.
